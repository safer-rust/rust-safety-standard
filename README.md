## About Rust Safety Standard
This standard aims to provide guidelines for unsafe code usage and documentation in Rust. See the draft [here](rust-safety-standard.md).

The standard is currently under discussion ons [Rust internals](https://internals.rust-lang.org/t/pre-rfc-rust-safety-standard/23963).
